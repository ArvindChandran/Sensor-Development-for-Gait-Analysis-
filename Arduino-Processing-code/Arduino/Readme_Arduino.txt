Instructions -

1) Place the contents of the libraries folder in location  C:\Users\User_Name\Documents\Arduino\libraries

2) Upload the Plantar_MPU_Multiple code onto the Arduino Board via USB cable ("Note: Check COM port")

3) Connect the Arduino board, Sensors to the buit cicuit and power via Power bank (5V)

4) Pair Bluetooth of the computer in use to onboard Bluetooth module (HC05)

5) Open Processing Folder and use the files accordingly
